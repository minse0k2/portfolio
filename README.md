# portfolio
저는 국어학을 깊이 연구하여 학문적 성취를 이루는 전문 연구자가 되고자 합니다. 
국어국문학을 전공하며 개별 언어 현상 속에 숨겨진 체계적인 규칙을 찾아내는 과정에서 큰 매력을 느꼈습니다. 
제가 지향하는 연구는 단순한 이론적 지식의 습득을 넘어, 언어가 사회 제도와 인간의 사고에 미치는 영향을 심도 있게 이해하고 이를 논리적으로 증명할 수 있는 학술적 역량을 배양하는 것입니다.

현재 저는 ‘국어학부’의 부장으로서 매주 학술 세미나를 기획하고 운영하고 있습니다. 
이 과정에서 교과서적 이론에 머물지 않고, 스스로 연구 문제를 정의하여 자료를 분석하는 데 집중했습니다. 
최근 진행한 세미나에서는 ‘법조문 표현의 특수성과 언어적 필요성’을 주제로 심도 있는 토론을 이끌었습니다. 
일반적인 문체와 달리 엄격한 정의와 제약이 요구되는 법률 문장의 특수성을 고찰하며, 법적 명확성을 위해 사용되는 특정 어휘와 문장 구조의 기능적 필연성을 분석했습니다. 
이러한 경험은 텍스트의 정밀한 구조를 파악하는 분석력을 높이는 계기가 되었으며, 연구자로서 갖춰야 할 비판적 사고력을 한층 강화해주었습니다.

앞으로 저는 이러한 연구자적 기초를 바탕으로, 국어학의 전통적 이론과 현대적 텍스트 분석론을 접목한 독창적인 연구 체계를 구축하고자 합니다. 
급변하는 사회적 환경 속에서 언어가 어떻게 규범성을 유지하며 기능하는지 추적하고, 이를 학계뿐만 아니라 실제 언어 정책이나 공공 담론의 질적 향상에 기여하는 실천적 연구로 확장하겠습니다. 
끊임없이 스스로 질문을 던지고 증명해 나가는 집요함을 유지하며, 국내외 학술 공동체와의 활발한 교류를 통해 국어학의 지평을 넓히는 연구자로 성장하겠습니다.
