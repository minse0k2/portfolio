# portfolio
저는 국어학 및 현대문학을 깊이 있게 탐구하여 학문적 성취를 이뤄낼 수 있는 연구자가 되고자 합니다.
국어국문학을 전공하며 언어의 체계와 문학적 텍스트를 분석하는 과정에서, 현상 너머의 원리를 규명하는 일에 큰 흥미를 느꼈습니다.
현재는 학술부인 '국어학부'의 부장으로서 매주 세미나를 기획하고 운영하며 실제 연구에 필요한 기초 체력을 쌓고 있습니다.
단순히 교과서적인 지식을 습득하는 것에 그치지 않고, 매주 발제문을 작성하고 부원들과 토론하며 하나의 주제를 심도 있게 파고드는 경험은 제가 학문의 길로 나아가는 데 확신을 주었습니다
학술부를 이끌며 가장 우선시하는 태도는 '자료에 근거한 비판적 사고'입니다.
세미나를 준비하다 보면 명확히 저의되지 않은 개념이나 해석이 갈리는 지접을 마주하게 됩니다.
저는 이를 가볍게 넘기지 않고 관련 논문과 문헌 자료를 찾아보며 논리적인 근거를 확보하는데 집중했습니다.
이러한 과정은 세미나 중 발생하는 다양한 질문에 명확한 방향을 제시하는 밑바탕이 되었고, 부원들과 함께 수준 높은 결론을 도출하는 동력이 되었습니다.
스스로 철저히 준비하는 자세가 연구 공동체의 논의를 얼마나 풍성하게 만들 수 있는지 깨달은 소중한 경험이었습니다.\
